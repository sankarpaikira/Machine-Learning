# PySpiders ML Practice

This repository contains machine learning practice notebooks and datasets for learning regression, encoding, scaling, and introductory ML concepts.

## Contents

- `datasets/` - sample CSV files used for training and practice
- `Linear_Regression/` - notebooks for simple and multiple linear regression
- `Polynomial_Regression/` - notebooks for non-linear regression examples
- `Logistic_Regression/` - notebook(s) for logistic regression concepts
- `ML_INTRO/` - introductory machine learning notebooks
- `Scaling & Encoding/` - notebooks for data preprocessing with scaling and encoding
- `requirements.txt` - Python dependencies for this project

## Usage

1. Clone the repository
2. Install dependencies: `pip install -r requirements.txt`
3. Open a notebook in Jupyter or VS Code
4. Practice daily and commit your changes regularly
